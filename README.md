# Marketing Funnel & Conversion Performance Analysis

## Task Information
- Track: Data Science & Analytics (DS)
- Task: Task 3
- Organization: Future Interns

## Objective
Analyze marketing funnel performance to understand user drop-offs
from View → Cart → Purchase stages and improve conversion rates.

## Dataset
E-Commerce User Behavior Dataset (Kaggle)

## Tools Used
- Power BI
- Microsoft Excel

## KPIs
- View Users
- Cart Users
- Purchase Users
- Funnel Conversion Analysis

## Key Insights
- High engagement at View stage (18.25K users)
- Major drop-off from View → Cart
- Very few users reach Purchase
- Conversion does not vary significantly by price range

## Recommendations
- Improve product CTAs and trust signals
- Offer cart-level incentives
- Simplify checkout flow
- Retarget high-intent users
